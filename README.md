 DropVault

DropVault is a Chrome Extension built to streamline tab management during research-heavy workflows. Whether you're comparing products, collecting resources, exploring supplier websites, or simply managing large numbers of browser tabs, DropVault provides a quick and organized way to save, access, and manage important links directly from the browser.

The project was developed using Vanilla JavaScript, Chrome Extension APIs, and Local Storage, with a focus on simplicity, speed, and usability.
